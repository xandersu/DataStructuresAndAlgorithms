# 第4章 堆和堆排序

普通队列：先进先出、后进先出

优先队列：出队顺序和入队顺序无关；和优先级相关

关键词：动态



|          | 入队    | 出队    |
| -------- | ------- | ------- |
| 普通数组 | O(1)    | O(n)    |
| 顺序数组 | O(n)    | O(1)    |
| 堆       | O(logn) | O(logn) |



对于总共N个请求：

使用普通数组或者顺序数组，最差情况：O(n^2)

使用堆：O(nlgn)



## 堆的基本实现

### 二叉堆 Binary Heap

二叉堆是一颗完全二叉树。（最大堆：越往上越大，但不一定）

堆中某个节点的值总是不大于其父节点的值。
