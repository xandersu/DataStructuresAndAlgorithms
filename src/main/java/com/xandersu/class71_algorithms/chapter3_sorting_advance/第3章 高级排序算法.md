## 归并排序 Merge Sort

首先分成一半

把左边的数组排序，把右边的数组排序

之后再将它们归并起来

分成一半后在分成一半，然后先排序再归并

再分半再归并

到一定细粒度后，就剩一个元素，不用排序就是有序的

然后进行归并，逐层归并



使用一块同样大的另外空间



时间复杂度O(nlogn)

O(N)的额外空间
