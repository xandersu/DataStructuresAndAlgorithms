# 回溯解题法

```
private 返回值 =  new 返回值;

private 使用状态;

主方法(要遍历的数据){
    数据校验
    new 使用状态;
    new 已经遍历的中间数据;
    调用 递归方法(要遍历的数据，0，已经遍历的中间数据)
    return 返回值;
}

递归方法(要遍历的数据，遍历到哪里，已经遍历的中间数据){
    //递归停止条件
    if(index == 要遍历的数据的数量){
        返回值 添加 已经遍历的中间数据
        return;
    }

    //获得正在遍历的数据，可选
    正在遍历的数据 = 要遍历的数据.get(遍历到哪里);
    for(int i=0;i<要遍历的数据的数量;i++){
         该位置 使用状态 置为已使用
         已经遍历的中间数据  增加 正在遍历的数据
         调用 递归方法(要遍历的数据，遍历到哪里+1，已经遍历的中间数据)
         该位置 使用状态 置为未使用
         已经遍历的中间数据  减少 正在遍历的数据  
    }
}
```
